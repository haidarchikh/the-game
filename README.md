## 0x02: The Runtime Environment ("The Lands")

Strict categorization of signals is required to prevent "Meaning" from overwriting "Reality."

- **Biology-land (The Hardware)**: Raw sensor data (heat, cold, tightness, vibration). Ideology-free.
- **Emotional-land (The Sensation)**: The system buffer. Holds raw affect (e.g., "I feel stupid").
- **Narrative-land (The Meaning-Making)**: Attaches "meaning" to data. Computationally expensive; prone to hallucinating threats.
- **Observer-land (The Profiler)**: Second-order cybernetic loop. Records the system observing itself.
- **Kernel-land (The Validator)**: The root process. Runs the ultimate check: `IS_OPERATIONAL?`
